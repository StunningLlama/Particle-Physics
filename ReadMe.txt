=== Wind Tunnel ===

Key Action
A	Brush mode: Add
R	Brush mode: Replace
D	Brush mode: Delete
M	Brush mode: Move (default)

W	Material: Water (default)
N	Material: Air
B	Material: Barrier

P	Pause
S	Save from file
L	Load from file
C	Reset simulation
F	Advance frame

1	Brush size: 1
2	Brush size: 2
3	Brush size: 3 (default)
4	Brush size: 4
5	Brush size: 5

9	Change background look (Blank/Pressure)
0	Change particle look (Material/Pressure/Invisible)

+	Increase density
-	Decrease density

]	Increase flow velocity
[	Decrease flow velocity

X	Special function
